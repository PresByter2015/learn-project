# 线性代数本质
```
[       ]        [         ]         [                      ]
  a  b              e   f                ae + bg   af + bh  
            *                   =  
  c  d              g   h                ce + dg   cf + dh
[       ]        [         ]         [                      ]


=>

[       ]        [      ]           [      ]           [      ]         [            ]
  a  b              e                  a                   b               ae + bg
            *                 =    e             +   g              =   
  c  d              g                  c                   d               ce + dg 
[       ]        [     ]            [      ]           [      ]         [            ]


=> 

[       ]        [      ]           [      ]           [      ]         [            ]
  a  b              f                  a                   b               af + bh
            *                 =    f             +   h              =   
  c  d              h                  c                   d               cf + dh 
[       ]        [     ]            [      ]           [      ]         [            ]


```
